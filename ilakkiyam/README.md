# இலக்கியம் | The Literature Shelf

Real text, read closely — the other half of every lesson. Each deep-dive gives மூலம் (source), சொற்பொருள் (word gloss), பொழிப்பு (plain rendering), நுட்பம் (craft), and எண்ணத்திற்கு (what it gives this project).

## காலவரிசை | The Map by Period

| Period | Works | Character |
|---|---|---|
| **சங்கம்** ~300 BCE–300 CE | எட்டுத்தொகை (8 anthologies) + பத்துப்பாட்டு (10 long poems): 2,381 poems, 473 named poets (~30 women, 102 poems anonymous) | Secular, precise, sensuous; அகம் (interior/love) and புறம் (exterior/war, praise, wisdom) |
| **சங்கம் மருவிய** ~300–600 CE | பதினெண்கீழ்க்கணக்கு incl. **திருக்குறள்**; caution: dating ranges vary | Ethical compression; aphorism |
| **காப்பியம்** ~450–900 CE | சிலப்பதிகாரம், மணிமேகலை (+ three mostly lost) | Narrative epics; Jain/Buddhist milieu |
| **பக்தி** ~600–900 CE | தேவாரம், திருவாசகம், நாலாயிரம் | Devotion; god addressed in the grammar of love |
| **இடைக்காலம்** ~900–1600 | கம்பராமாயணம், commentaries, நன்னூல் | Court epic; the great உரை tradition |
| **இக்காலம்** 1600– | Prose birth → first novel (1879) → **பாரதி** → modern fiction, Eelam & diaspora writing | Prose, nation, self |

**A note on "Sangam prose":** there is none — the classical corpus is entirely verse. Tamil's ancient voice is compression and music; sustained prose is a 17th–19th century development (logged as எ-005). What the age offers instead of prose is a poetics so codified it functions like an *engineering spec* for expression — see the திணை table in `sangam.md`.
