# ஆய்வு | The Research Track

The question this track carries: **where does Tamil strain when a deeply realized thought passes through it — and how does it grow past that?**

## முறை | Method

1. **காண் — observe.** In lessons, reading, and daily thought, notice where expression loses something: a nuance with no word, a structure that forces a false shape, an experience the grammar cannot hold. Log it in `ellai.md`.
2. **விதை — seed.** Notice equally where Tamil already outruns other languages — distinctions it makes that they cannot. Log in `vithai.md`. Seeds are both proof that engineered expression is possible and raw material for it.
3. **வளர் — grow.** From limits and seeds, propose evolutions: coinage, revival, structural extension. A proposal counts as language only when it carries thought between minds without loss — tested in use, between us, in these files.

Rule of the track: **no proposal without a logged limit it answers.** Evolution stays honest by staying downstream of observed need.

Phase 3 (அப்பால்) inherits the [Koine](https://github.com/ilamgumaran/thought-org-with-human-ai-hybrid/tree/main/proposed-repos/koine) work when this track reaches it.
