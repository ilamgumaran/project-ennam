# பாடம் | The Learning Track

Starting point: fluent speech, script and grammar being built. So Tamil is not learned here as a foreigner would learn it — we attach writing and structure to a language already alive in the speaker.

## வரிசை | The Spine

The classical tradition divides grammar five ways (ஐந்திலக்கணம்); it is our syllabus spine, in Tamil's own categories:

1. **எழுத்து** — letters: the script system, its sounds, its combination rules
2. **சொல்** — words: how words are built, inflected, classified
3. **பொருள்** — meaning: semantics and the poetics of content (அகம்/புறம், திணை)
4. **யாப்பு** — prosody: meter, the music of the line
5. **அணி** — ornament: rhetoric and figures

Literature is woven in from the first lesson, not saved for the end — each lesson closes with real text, and the deep readings live on the shelf: `../ilakkiyam/`. History runs alongside in `../varalaru/`. Every lesson also feeds the research track (`../aaivu/`): what Tamil said better than English → `vithai.md`; what it could not yet say → `ellai.md`; patterns and essence → `saaram.md`.

## படிமுறை | How Difficulty Rises

Lessons 01–03 are deliberately simple — one big idea each, drills a fluent speaker can run from speech. From 04 the load rises: full systems (all eight cases, புணர்ச்சி rules), then classical text read in the original with decreasing scaffolding, then composition — first prose, then verse under யாப்பு rules. The ramp: **recognize → build → read → compose.** A learner finishing the track should be able to read a Sangam poem with a gloss, and write a correct வெண்பா line — and to *appreciate why both matter*.
